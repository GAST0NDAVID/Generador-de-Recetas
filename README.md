# Generador de Recetas

Proyecto estático que permite generar recetas aleatorias y visualizar detalles como ingredientes y pasos.

## Características
- Generación aleatoria de recetas desde una lista interna.
- Diseño moderno y minimalista.
- Interfaz completamente estática sin base de datos.

## Estructura principal
- **components/**: Componentes de la interfaz.
- **lib/recipes.ts**: Fuente de datos de recetas.
- **public/**: Imágenes e íconos.
- **styles/globals.css**: Estilos globales.

## Requisitos
- Node.js
- pnpm o npm

## Instalación
```bash
pnpm install
pnpm dev
```
